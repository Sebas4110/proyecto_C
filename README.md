# Proyecto_C — Calculadora Científica

Calculadora científica avanzada que se ejecuta en terminal.

## Requisitos

- Python 3.10 o superior
- Git

## Cómo ejecutar

1. Clona el repositorio
   git clone https://github.com/Sebas4110/proyecto_C.git

2. Entra a la carpeta
   cd Proyecto_C

3. Ejecuta el script
   - Windows: start.bat
   - Mac/Linux: bash start.sh

## Módulos

- Operaciones básicas
- Operaciones avanzadas
- Trigonometría
- Estadística
- Geometria
